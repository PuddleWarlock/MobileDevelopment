### **1. Задание: Работа с ScrollView**

Был создан отдельный модуль `ScrollViewApp`. В соответствии с заданием реализовано отображение геометрической прогрессии со знаменателем 2 от 1 до 100-й степени. Элементы добавляются динамически внутри `LinearLayout`, помещенного в `ScrollView`.

**Файл `MainActivity.java` (модуль ScrollViewApp):**
```java
LinearLayout wrapper = findViewById(R.id.wrapper);
for (int i=1; i <= 100; i++){
    View view = getLayoutInflater().inflate(R.layout.item, null, false);
    TextView text = (TextView)view.findViewById(R.id.textView);
    // Вычисление степени двойки
    text.setText(new StringBuilder().append("Item ").append(Math.pow(2,i)).toString());
    wrapper.addView(view);
}
```

### **2. Задание: Работа с ListView**

Создан модуль `ListViewApp`. Реализован список из 30 книг и их авторов. Использован стандартный макет `android.R.layout.simple_list_item_2` для отображения двух строк текста (название и автор) в одном элементе списка. Для адаптации данных использован переопределенный `ArrayAdapter`.

**Файл `MainActivity.java` (модуль ListViewApp):**
```java
private Map<String, String> books = Map.ofEntries(
        entry("Война и мир", "Лев Толстой"),
        entry("Преступление и наказание", "Фёдор Достоевский"),
        // ... еще 28 книг
);

// Адаптер для двухстрочного списка
ArrayAdapter<Map.Entry<String, String>> adapter = new ArrayAdapter<>(this,
        android.R.layout.simple_list_item_2, android.R.id.text1, listData) {
    @NonNull
    @Override
    public View getView(int position, View convertView, @NonNull ViewGroup parent) {
        View view = super.getView(position, convertView, parent);
        TextView text1 = view.findViewById(android.R.id.text1);
        TextView text2 = view.findViewById(android.R.id.text2);

        Map.Entry<String, String> entry = getItem(position);
        if (entry != null) {
            text1.setText(entry.getKey());   // Название книги
            text2.setText(entry.getValue()); // Автор
        }
        return view;
    }
};
ListView countriesList = findViewById(R.id.country_list_view);
countriesList.setAdapter(adapter);
```

### **4. Контрольное задание: Интеграция списка в основное приложение**

В основном проекте `PocketOrnithology` реализован экран "Журнал наблюдений" (`JournalFragment`) с использованием `RecyclerView`.

1.  **ViewModel (`JournalViewModel`):** Загружает список наблюдений из базы данных Room через UseCase (`GetJournalUseCase`) и передает их в LiveData.
2.  **Adapter (`ObservationAdapter`):** Отображает фото птицы (загружается с диска), название, дату и местоположение. Реализован интерфейс клика для перехода к деталям.

**Файл `JournalFragment.java` (связывание):**
```java
RecyclerView recyclerView = view.findViewById(R.id.recycler_view);
recyclerView.setLayoutManager(new LinearLayoutManager(requireContext()));

// Настройка адаптера и обработки кликов
ObservationAdapter adapter = new ObservationAdapter(observation -> {
    // Открытие фрагмента деталей
    ObservationDetailFragment detailFragment = ObservationDetailFragment.newInstance(observation.id);
    getParentFragmentManager().beginTransaction()
            .replace(R.id.fragment_container, detailFragment)
            .addToBackStack(null)
            .commit();
});
recyclerView.setAdapter(adapter);

// Подписка на данные из БД
viewModel.getObservations().observe(getViewLifecycleOwner(), list -> {
    if (list.isEmpty()) {
        // Показ заглушки, если список пуст
    } else {
        adapter.setObservations(list);
    }
});
```

**Файл `ObservationAdapter.java` (отображение элемента):**
```java
public void bind(Observation item, OnItemClickListener listener) {
    name.setText(item.birdName);
    SimpleDateFormat sdf = new SimpleDateFormat("dd.MM.yyyy HH:mm", Locale.getDefault());
    date.setText(sdf.format(new Date(item.timestamp)));

    // Отображение фото наблюдения
    if (item.photoPath != null) {
        File imgFile = new File(item.photoPath);
        if (imgFile.exists()) {
            Bitmap myBitmap = BitmapFactory.decodeFile(imgFile.getAbsolutePath());
            image.setImageBitmap(myBitmap);
        }
    }
    itemView.setOnClickListener(v -> listener.onItemClick(item));
}
```
